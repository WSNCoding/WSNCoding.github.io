Anagrams Installation Guide

When asked for installaton path we recommend selecting your desktop: C:\Users\INSERTUSERNAME\Desktop\Anagrams

This will create a file on your desktop called Anagrams. To create a shortcut for ease of use, enter the folder and right click the .exe file. Then select send to, then Desktop. Further extensive guide within install folder (README.txt).
